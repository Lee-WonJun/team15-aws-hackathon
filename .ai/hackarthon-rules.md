# Amazon Q Developer 해커톤 개발 규칙

## 필수 요구사항
- AWS 서비스 최소 1개 이상 사용 필수
- IaC (Terraform, AWS CDK 등) 사용 필수, 해당 코드도 제출물에 포함

## 제출물
- Git Repository (코드)
- 발표 Presentation (데모 영상 + 핵심 프롬프트 포함)

## 평가 기준
- Amazon Q Developer 활용도 및 창의성 (35%)
- 비즈니스 가치 및 실용성 (30%)
- 기술적 구현 완성도 (20%)
- 발표 및 데모 (15%)

## 주의사항
- 발표 시 핵심 프롬프트 공유 필수
