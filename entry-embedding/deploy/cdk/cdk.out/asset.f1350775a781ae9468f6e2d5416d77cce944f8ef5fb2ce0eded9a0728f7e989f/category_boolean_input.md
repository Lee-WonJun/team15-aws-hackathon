# boolean_input 카테고리 블록들

## is_clicked
**Python 문법:** `Entry.is_mouse_clicked()`

**사용 예시:**
```python
Entry.is_mouse_clicked()
```

## is_press_some_key
**Python 문법:** `Entry.is_key_pressed(%1)`

**매개변수:**
- VALUE: 위치 0

**사용 예시:**
```python
Entry.is_key_pressed(%1)
```
